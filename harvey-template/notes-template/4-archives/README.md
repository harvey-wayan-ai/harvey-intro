## Archiving Workflow

### When to Archive?
1. **Projects:** When completed or abandoned
2. **Notes:** When no longer relevant/needed
3. **Resources:** When outdated or superseded

### How to Archive?
1. Move file/folder to appropriate year/category in archives
2. Update any links in active notes
3. Add completion date to filename: `project-name-COMPLETED-2026-01-27.md`
4. Optional: Add summary in archive index
